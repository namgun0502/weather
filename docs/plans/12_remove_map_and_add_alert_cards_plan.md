# 기상 지도 제거 및 기상 특보 카드 UI 전환 계획서

## 1. 목적 및 배경
- 기존 D3.js 기반 한반도 지도 렌더링에서 지역 폴리곤 겹침 오류, 지도가 전부 단색으로 칠해지는 문제, 외부 GeoJSON 로드 불안정성 등의 오류가 잦아 사용자 경험을 저해함.
- 복잡하고 인식이 어려운 지도 UI 대신 직관적인 **5개 기상 특보 카드(기온, 미세먼지, 폭염, 호우, 폭설)** 및 **전국 주요 도시 현황판** 형태로 전면 개편하여 날씨 정보를 한눈에 볼 수 있도록 조치함.

## 2. 주요 개선 기능
- **기상 특보 현황판 도입:** 선택한 지역의 5대 핵심 기상 수치(기온, 미세먼지, 폭염, 호우, 폭설)를 한눈에 볼 수 있는 개별 정보 카드로 구성.
- **전국 주요 도시 현황판:** 서울, 부산, 대구, 인천 등 9대 광역 권역의 기상 특보 상태(폭염, 호우, 폭설 등)를 색상 배지 형태로 요약 노출.
- **연동 버그 해결:** 
  - 검색창에서 찾은 좌표와 미리 등록된 퀵 버튼의 좌표가 미세하게 달라 결과가 달라지던 문제를 `FIXED_CITY_COORDS`로 매핑하여 일치시킴.
  - 미세먼지 계산에 적용된 무작위 난수 요소(`Math.random`)를 완전히 배제하여 동일 날씨 조건에서는 100% 동일한 미세먼지 값이 나오도록 조치.

## 3. 관련 파일 목록
- `weather/index.html`: 지도 영역 제거 및 특보 카드 그리드, 전국 주요 도시 테이블 HTML 추가
- `weather/style.css`: 지도 관련 스타일 완전 삭제 및 특보 카드, 배지, 주요 도시 그리드 스타일 추가
- `weather/app.js`: GeoJSON 로직 및 지도 드로잉 코드 완전 제거, 특보 데이터 계산 및 전국 현황판 데이터 바인딩 로직 개발
- `docs/plans/12_remove_map_and_add_alert_cards_plan.md`: 본 이력 관리 문서
